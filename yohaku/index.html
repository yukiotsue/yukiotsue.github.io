Introduction

はじめに

視覚デザインにおける余白は、視覚的バランスや情報伝達において重要な役割を果たすとされている。筆者らは前報において描線ベースの手法で余白抽出実験を実施したが、手法上の課題が生じた。本研究ではこれを克服するグリッドベースの余白抽出手法を提案・実装し、検証実験によって余白知覚の傾向を検討した。

前提：知覚的余白とは
余白は「何も描かれていない物理的な空白」として一義的に定義できるわけではない。村田ら（2016）が示した通り、人が知覚する余白は画像上の物理的な空白と必ずしも一致しない可能性がある。したがって余白を定量的に捉えるには、物理的な空白領域の検出ではなく、人の感性的・知覚的な判断にもとづいて余白領域を直接抽出する手法が必要となる。

前報・描線ベース手法の3つの課題

1

描画粒度のばらつき

広い余白を単一の描線で囲むスタイルと、細かく分けて囲むスタイルが混在し、同一領域でも抽出結果に差異が生じた

2

境界の曖昧さ

フリーハンドの線の内外が不明瞭となり、余白領域の境界を一意に定義できなかった

3

操作環境の不統一

大半の参加者がトラックパッドを使用し、入力精度のばらつきにより意図しない選択・選択漏れが発生した

同一ポスターに対する参加者ごとの描画スタイルのばらつき

Proposed Method

グリッドベースツールの提案

上記の課題に対し、画像を均等なグリッドに分割してセル単位で選択する手法を提案した。セルの選択・非選択の二値的判断によって境界の曖昧さを排除し、同一仕様の機材で操作環境を標準化することを試みた。グリッド解像度は 10×10・20×20・30×30 の 3 パターンを比較検討し、操作性と測定精度のバランスから 20×20（400 セル） を採用した。

ツールの仕様
端末 iPad（第 7 世代, 10.2 インチ）+ Apple Pencil
入力設計 Apple Pencil = 選択 / 指ジェスチャー = ズーム・パン
描画モード ペンモード / 消しゴムモード（Undo 機能あり）
出力形式 画像（HEIC / JPEG）+ JSON
Validation Experiment

検証実験

実験の設計
参加者 22 名（大学生・大学院生、全員がデザイン関連科目を履修）
刺激 架空の鉄道会社による観光広告ポスター 8 点（グレースケール変換・色彩の影響を排除）
刺激の選定 前報の余白分布データを四分位数で 4 段階に分類し、各段階から代表的なレイアウト特徴を持つ 2 点ずつを参考に制作
手続き 「余白を感じる領域」をグリッド単位で選択（余白の定義は示さず、直感的な判断を求めた）
倫理審査 筑波大学芸術系研究倫理委員会の承認を得て実施
Results

結果

余白選択の基本統計量

各ポスターの平均選択グリッド数は No.6 の 44.00 から No.4 の 167.09 まで大きく異なり、デザインによって知覚される余白量に差があることが示された。変動係数（CV）による参加者間一致度の分析では、No.4 が CV = 0.308 で最も高い一致を示し、No.7 は CV = 0.948 で最も低い一致度を示した。

余白知覚の4つの特徴的傾向

全参加者の選択領域を合成したヒートマップの分析から、以下の傾向が認められた。これらは村田ら（2016）が示した知覚的余白の空間特性と概ね一致するものであった。

1
オブジェクトの周囲は余白として知覚されやすく、オブジェクト自体は余白として選択されなかった
2
画像の端部・角および単色・均一な背景領域は余白として知覚されやすかった
3
オブジェクト密度が高い領域やテキストが配置された領域は余白として知覚されにくかった
4
立体感・奥行き感が余白知覚に影響を及ぼし、奥行きのある背景は余白として選択されにくい場合があった

ツールの操作性評価

実験後の質問紙調査において、ツールの操作性は全項目で平均 4.0 以上（5 段階評価）の評価を得た。一方で、グリッド解像度の限界（字間等の微細な領域の選択困難）や、グリッドが境界を跨ぐ場合の判断の難しさも指摘された。

Discussion & Conclusion

考察・まとめ

一致度に影響する3つの視覚的要因

実験結果から、余白知覚の一致度に影響する視覚的要因として以下の 3 点が示された。コントラストが高く背景が均一なポスターでは参加者間の判断が一致しやすく、写真背景を持つポスターでは知覚が大きく分かれた。

① コントラスト（明度差） 高コントラスト → テキスト・余白の境界が明瞭 → 一致しやすい
② 背景の均一性 均一な背景 → 「情報なし＝余白」の判断が容易 → 一致しやすい
③ 写真背景の有無 写真背景 → 「1 つのオブジェクト」vs「明るい部分＝余白」に二極化

余白知覚の暫定的な3類型

CV 値と判断基準の対応から、余白知覚の暫定的な類型化が示唆された。ただし参加者は複数の基準を複合的に用いており、この類型化は暫定的な試みとして今後の検証が必要である。

CV < 0.4
物理的特徴に
もとづく余白
客観的な空白として知覚。参加者間で一致しやすい
CV 0.4 – 0.7
視覚的知覚に
もとづく余白
視覚的評価を伴い、解釈の余地がある
CV > 0.7
意味的解釈に
もとづく余白
デザイン意図の推測を要し、主観性が高い
まとめと今後の課題

本研究では、描線ベースツールの課題を克服するグリッドベースの余白抽出手法を提案・実装し、22 名による検証実験を実施した。その結果、余白知覚の一致度がポスターの視覚的特性（コントラスト・背景の均一性・写真背景の有無）に依存する傾向が示された。開発したツールは操作性においても高い評価を得た一方、グリッド解像度の限界や境界判断の困難さなどの改善課題も明らかとなった。
